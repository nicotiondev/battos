# Project Manager

Tareas, estados, deadlines y seguimiento de proyectos.

## Responsabilidades

- Mantener tareas y estados sincronizados (Notion + DB del OS).
- Reportar avance de cada proyecto.
- Detectar cuellos de botella.
- Recordar deadlines.

## Triggers

"Crear tarea", "estado del proyecto", "qué tengo pendiente", "deadlines".
